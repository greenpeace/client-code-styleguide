3.1 Generic Form Item

Labels and input are wrapped in a div with the type of the class.

Compatible in FF2+, Chrome1+, IE?, Opera?, Safari?

Since version 3.0.

:hover

subtle hover highlight.

:focus

green border.

:disabled

grey out to indicate it cannot be used.

<form action="#" id="action-form-pledgeName" method="post" accept-charset="utf-8" class="js-track-abandonment">
  <div class="input text email">
    <label for="UserEmail">Email</label>

</div>

3.3 Labels

Label and input are wrapped in div. If required class is added, additional span with a @red-1 star is included.

.show-default

Default mode of showing the labels.

.horizontal-ra

Show labels in left side with right text alignment.

.horizontal-la

Show labels in left side with left text alignment.

.inline

Show label and input inline.
